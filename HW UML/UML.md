## Sequence

```plantuml
@startuml

mainframe **sd** Заказ кредитной карты 
actor Клиент as Cl
participant МП as App
database Eq as S
database "Кредитный конвейер КК" as CC
database "Скоринг система" as SS
database Каталог as Cat
database Отделения as Deps

Cl -> App : Авторизация в мобильном приложении
activate Cl
activate App
App --> Cl : Пароль введен успешно
App --> Cl: Приветственный экран
Cl -> App : Клик на "Оформить кредитные карту"
App -> S : Запрос на финансовые показатели клиента
S --> Cat : Сформированный "портрет" клиента
Cat --> App : Формирование подборки кредитных предложений для клиента
App --> Cl : Отображение доступных кредитных предложений
App ->> SS : Проверка клиента на кредитоспособность
Cl -> App : Выбор предложения из доступных вариантов
App --> Cl : Создание заявки на открытие карты
SS --> CC : Завершение процесса проверки клиента на кредитоспособность
CC --> App : Передача ответа о возможности получить кредитную карту
opt Отказ в выдаче кредитной карты
App --> Cl : Ответ о неготовности предоставить кредитную карту
App --> Cl : Возврат в меню
end
App --> Cl : Подтверждение готовности предоставить кредитную карту
Cl -> App : Выбор способа получения карты
App --> Cl : Предоставление возможности доставки курьером или забрать в отделении
Cl -> App : Выбор забрать из отделения
App -> Cat : Запрос на список ближайших доступных отделений по геолокации клиента
Cat --> App : Формирование списка отделений
App --> Cl : Выдача списка доступных отделений
Cl -> App : Выбор отделения
App --> Cl : Предложение выбрать дату и время для готовности карты
Cl -> App : Выбор даты и времени
App -> Cl : "Заявка на оформление карты успешно завершена"
App -> S : Сохранение информации об оформлении кредитной карты
S -> Deps : Отправка заявки на создание кредитной карты
Deps --> Cl : Готовность выдать карту в указанные клиентом дата/время
@enduml
```
